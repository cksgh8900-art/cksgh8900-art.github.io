# 첫 번째 게시글

이것은 GitHub Pages 정적 블로그의 샘플 게시글입니다.

## 소개

이 블로그는 다음과 같은 기술을 사용합니다:

- HTML, CSS, Vanilla JavaScript
- marked.js를 사용한 마크다운 파싱
- Prism.js를 사용한 코드 하이라이팅
- Giscus를 사용한 댓글 시스템

## 코드 예시

다음은 JavaScript 코드 예시입니다:

```javascript
function greet(name) {
    return `Hello, ${name}!`;
}

console.log(greet('World'));
```
